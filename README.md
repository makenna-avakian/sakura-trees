## Installation  
1. Search **`Moonlit Cherry Blossoms`** in VSCode Extension Marketplace.  
2. Install the theme and reload your VSCode.  
3. Press **F1**, search `"Preferences: Color Theme"`, and select **Moonlit Cherry Blossoms**.
